**Monte Carlo Pi Approximation**

This is a simple demonstration of a Monte Carlo method to approximate the value of pi.

Usage:

It runs as a simple python script and can be run directly from terminal. Iteration input must be a positive integer.
If you have to adjust the batch size, you will have to do so in the code by adjusting the batch_size variable.

Dependencies:

This file assumes numpy has been installed in the global environment.
